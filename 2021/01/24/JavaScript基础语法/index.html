JavaScript基础语法

变量

大小写敏感

JavaScript 的变量名区分大小写，A和a是两个不同的变量

默认值undefined

如果只是声明变量而没有赋值，则该变量的值是undefined。undefined是一个特殊的值，表示“无定义”

弱类型

JavaScript 是一种动态类型语言，即变量不用指定具体的数据类型，也可以随时更改类型

变量赋值可以省略var，但这种做法不推荐

变量提升

JavaScript 引擎的工作方式是，先解析代码，获取所有被声明的变量，然后再一行一行地运行。这造成的结果，就是所有的变量的声明语句，都会被提升到代码的头部，这就叫做变量提升（hoisting）

中文标识符

中文是合法的标识符，可以用作变量名，但这种做法不推荐

注释

单行注释：//

多行注释：/*...*/

此外，由于历史上 JavaScript 可以兼容 HTML 代码的注释，所以<!--和-->也被视为合法的单行注释

注意：这里的“和”意思是，<!--后的同一行内容会被注释掉，-->后的同一行内容也会被注释掉（注意<!--和-->不能有空格）

需要注意的是，-->只有在行首，才会被当成单行注释，否则会当作正常的运算

JS中的==与===

==：不是严格意义上的相等，比较的仅仅是两边变量的数值是否相等。当两边的值类型不同时，先进行类型转换再比较

注意：==比较数值时，如果其中至少一个是NaN（只能用isNaN()判断），那么一定不等

===：是严格意义上的恒等，首先比较两边的变量数据类型是否相等，然后再比较数值是否相等。不做类型转换，类型不同的一定不等

注意：switch结构中采用的是严格相等运算符（===），即不发生类型转换

标签（label）

JavaScript 语言允许，语句的前面有标签（label），相当于定位符，用于跳转到程序的任意位置

标签通常与break语句和continue语句配合使用，跳出特定的循环
