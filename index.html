git的下载及配置

git的下载安装

  • windows下直接在官网下载安装git for windows。下载安装后得到一个git bash工具，使用git bash来执行一系列的git命令。

  • 也可以选择先下载安装msys2，然后使用msys2提供的bash环境完成更多的东西，包括git的下载使用。其实，git for windows内置了一个msys2。

    使用msys2下载git：

    1. 下载msys2，由于墙的存在，选择官网下载会非常慢，推荐在开源镜像站找资源下载。

    4. 打开msys2，输入

    5. 安装git

    ps:
    安装后的git，你在本机的cmd窗口输出git version会提示不是内部或外部命令,也不是可运行的程序，那是因为msys2的虚拟环境中的path不是本机的path，相当于git是安装在虚拟环境的一个目录下，但这并不影响git的使用。而git bash的path用链接链到了你windows的users目录，当然这会引发一些问题。但是，我上传博客的时候还是使用的git bash，因为msys2下的生成的ssh目录路径并不是本机的，所以上传到服务器的时候会出现问题。

    git相关配置问题今后将会另外总结……

  • Nodejs
    官网下载好nodejs后安装即可。然后是配置nodejs。

  • github
    首先你要有一个github账号，然后创建repository，repository的名字是username.github.io(username是你账号的username，这个仓库名字就是搭建成功后博客的域名，所以尽量不要有一些奇怪的username)。

二、搭建博客

  1. 创建个人博客文件夹

三、本地git项目与github联动

SSH：简单来说就是一种加密的网络协议(Secure Shell)
Git是分布式的管理工具，而远程的代码管理是基于SSH的

  1. 检查本机SSH Key的设置

    显示“No such file or directory”，则进行下一步

  2. 生成新的SSH Key

    期间按一次回车，输入两次密码即可。
    设置的密码是提交项目时的密码，防止别人往项目里乱提交东西。

  3. 将SSH Key添加到Github
    找到id_rsa.pub文件，打开后复制文件里的内容。
    打开github主页，找到Settings，选择SSH and GPG keys，New SSH key,Title随便取一个名字(如：blog),Key中粘贴刚才文件里的内容，Add SSH key。

  4. 测试

    期间输入yes回车，输入之前设置的密码回车。
