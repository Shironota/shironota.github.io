Hexo中渲染LaTex

发表于 2018-12-23 | 分类于 Hexo

有时我们需要在Markdown中插入数学公式，而LaTeX是一种基于叫TeX的排版系统，所以我们需要进行一些设置，让我们的hexo博客能够正常渲染LaTex公式，让公式看起来更加优雅~

更换渲染引擎

默认渲染引擎是 hexo-renderer-marked，该引擎会把一些特殊的 markdown 符号转换为相应的 html 标签。hexo-renderer-kramed 引擎是在默认的渲染引擎的基础上修改了一些 bug ，两者比较接近，也比较轻量级。
